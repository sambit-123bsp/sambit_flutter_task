## setup for backend 

In this project you have to run the backend file locally.

- For run the backend file use yarn and yarn dev.
- Then put the ip address in the baseurl present in the api_services.dart

## setup for frontend

This project was built using Flutter version 3.32.5. Before getting started, make sure you have this version installed to ensure compatibility and avoid unexpected issues.
